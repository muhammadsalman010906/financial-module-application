Financial Module Application

A web-based financial management system developed using PHP and MySQL. This system helps manage income, expenses, users, and transactions in a structured way.

Features
- User registration and login system
- Add, edit, and delete transactions
- Income and expense tracking
- User management system
- Reports and audit logs
- Dashboard overview of financial data

Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

Project Structure
- accounts.php
- login.php
- register.php
- transactions.php
- users.php
- reports.php
- db.php
